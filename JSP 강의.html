스왑 영역(swap)

운영 체제 메모리를 데이터들 중에서 아주 가끔 읽거나 쓰기를 하는 데이터를 저장하는 디스크 영역
가상 메모리 용량 = 스왑 영역 용랑 + RAM 용량
가상 메모리에 할당된 디스크 영역을 스왑 영역이라고 한다.
스왑 영역 크기 = 1M * 1024 (1G)
메모리 용량, 사용량 조회
Bash
복사
ubuntu@:~$ free -m total used free shared buff/cache available Mem: 949 201 223 0 524 571 Swap: 0 0 0
​
디스크 용량 조회
Bash
복사
ubuntu@:~$ df Filesystem 1K-blocks Used Available Use% Mounted on /dev/root 7941576 2338472 5586720 30% / tmpfs 486012 0 486012 0% /dev/shm tmpfs 194408 876 193532 1% /run tmpfs 5120 0 5120 0% /run/lock /dev/xvda15 106832 6186 100646 6% /boot/efi tmpfs 97200 4 97196 1% /run/user/1000
​
스왑파일 생성
Bash
복사
ubuntu@:~$ sudo dd if=/dev/zero of=/swapfile bs=1M count=1024 1024+0 records in 1024+0 records out 1073741824 bytes (1.1 GB, 1.0 GiB) copied, 6.18846 s, 174 MB/s
​
스왑파일 권한부여
Bash
복사
ubuntu@:~$ sudo chmod 600 /swapfile ubuntu@:~$ sudo mkswap /swapfile Setting up swapspace version 1, size = 1024 MiB (1073737728 bytes) no label, UUID=610e7267-a854-42a1-9414-a0f90fa09bab ubuntu@:~$ sudo swapon /swapfile
